What is Virtual Bandung?

This is a virtual place which modeled Bandung city, Indonesia. We made this new system to travel and learn about Bandung. You can access easily and do sightseeing from your home like a RPG game. This game is made by team "Banja-i", who participated in SCUAD.

SCUAD

Four Fancy Places

There are four stages which reproduct real Bandung seeing spots. You can explore these places anytime,aniwhere and you don't have to whach the depature time. Each spot has original mision to which make your journey more fun. You can know about Bandung through garbage picking, crazy dance battle and so on.
